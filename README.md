## About:

Ocean Man presents the Brick Vault, a simple and local way to store and see which LEGO© sets you have in your collection.


## Documentation:

* **[Github](https://github.com/ocean-onion/BrickVault/tree/main/Docs)**


## Features:

* Add and manage owned and wanted Lego sets
* Display all sets in a formatted ASCII box layout
* Update any field on an existing set
* Remove sets by name or set number
* Change a set between owned and wanted
* Search by name, set number, theme, year, price range, piece count, built status and priority
* Count sets by theme, year, price range, piece count, built status and priority
* Data persisted locally via JSON with automatic split files for owned and wanted sets
